ul vs. ol

  • unordered list
    • HTML- tag: ul
    • liestet Punkte mit bullet Points auf
    • Beispiele:
      • To-Do Liste
      • ....
      • ....
    • weiterführende Links
  • ordered List
    • HTML-tag: ol
    • listet punkte mit Nummerieerungen auf
    • Beispiele:
      1. 1.Inhaltsverzeichnis
      2. ....
      3. ....
    • weiterführende Links
